 SmartBOMFilter - 智能物料筛选器
========================================
【这个工具能做什么？】
帮你自动清理 Excel BOM（物料清单）中的辅材，只保留核心零件。
例如：保留 “相机”，但删除 “相机线缆”、“相机支架”。
【核心规则：什么时候删除？】
如果物料名称【包含】“白名单词”，但【同时包含】该词对应的 “排除词”→ 删除。
举例：白名单词 “相机”，排除词 “线缆”。物料 “相机线缆”→ 删除。
如果物料名称【不包含任何白名单词】，但【包含】“全局黑名单词”→ 删除。
举例：全局黑名单有 “线缆”。物料 “普通线缆”（不含任何白名单词）→ 删除。
其他情况（包含白名单词且不含排除词，或不含任何关键词）→ 保留。
实际举例：
“相机” → 保留（命中白名单词，不含排除词）
“相机线缆” → 删除（命中白名单词 “相机”，同时包含排除词 “线缆”）
“工控机电源线” → 删除（命中白名单词 “工控机”，同时包含排除词 “电源线”）
“普通线缆” → 删除（不包含任何白名单词，但命中全局黑名单 “线缆”）
“CPU” → 保留（命中白名单词，无排除词）
【怎么使用？】
一、配置规则（打开软件后，左右两个区域）
左边：白名单词 + 排除列表
在下方输入框输入词（如 “相机”），点击 “添加白名单词”。
点击该词右侧的 “📝 排除列表”，在弹出的窗口中添加排除词（如 “线缆”）。
右边：全局黑名单
在下方输入框输入词（如 “支架”），点击 “添加黑名单词”。
所有规则会自动保存到 filter_config.json 文件，下次启动自动加载。
二、运行筛选
把需要处理的 Excel 文件（.xlsx）放在程序所在文件夹。
点击 “运行批量筛选” 按钮。
程序会自动处理每个文件，在同一个 Excel 里新增三个工作表：
“删减后”：保留的行（第一列会写明保留原因）
“被移除”：删除的行（第一列写明删除原因）
“源数据”：原始数据
三、生成汇总报告（可选）
点击 “生成汇总报告” 按钮，会在当前文件夹生成一个以文件夹名命名的 Excel 文件。
第一个工作表是文件列表，后面每个工作表对应一个原文件的 “删减后” 数据。
【关于 filter_config.json 配置文件】
该文件是工具的核心配置载体，自动生成 / 更新，存储所有筛选规则，格式为 UTF-8 编码的 JSON 格式，结构说明如下：
核心结构
json
{
    "whitelist": ["白名单词1", "白名单词2", ...],  // 所有白名单词的数组（自动排序）
    "whitelist_excludes": {                      // 白名单词对应的排除词映射
        "白名单词1": ["排除词1", "排除词2", ...],
        "白名单词2": [],  // 无排除词则为空数组
        ...
    },
    "global_blacklist": ["全局黑名单词1", "全局黑名单词2", ...]  // 全局黑名单词数组
}
关键说明
whitelist 数组与 whitelist_excludes 的 key 完全对应，程序会自动同步两者的内容（新增白名单词时自动在 whitelist_excludes 生成空数组，删除时同步删除）。
手动编辑：若熟悉 JSON 语法，可直接修改该文件（需确保格式正确），保存后重启工具即可生效；若格式错误，工具会加载失败并自动恢复默认配置。
默认配置：删除该文件后，工具下次启动会生成默认配置（包含相机、工控机、CPU 等基础规则）。
编码要求：文件必须为 UTF-8 编码，否则可能出现中文乱码或加载失败。
【注意事项】
程序会直接修改原 Excel 文件（新增工作表），建议提前备份。
只处理当前文件夹下的 .xlsx 文件，不会进入子文件夹。
Excel 文件的第一个工作表必须包含 D 列（物料名称列）。
如果不想保存配置，可以删除 filter_config.json 文件，下次启动会恢复默认规则。
手动修改 filter_config.json 后，需重启工具才能生效；工具运行中修改配置会实时保存到该文件。
【常见问题】
Q：我添加了一个新词，怎么确认它生效了？
A：添加后会自动跳转到该词所在页并高亮显示，也可查看 filter_config.json 确认词已写入对应字段。

Q：一个物料同时匹配多个白名单词怎么办？
A：取长度最长的白名单词进行判断。

Q：白名单词和全局黑名单词冲突怎么办？
A：只要命中白名单词（且不触发排除词），就不会被全局黑名单删除。

Q：修改了 filter_config.json 但工具没生效？
A：检查 JSON 格式是否正确（如逗号、括号是否匹配）、文件编码是否为 UTF-8，修改后需重启工具。

Q：filter_config.json 被误删了怎么办？
A：备份好或者找开发者再要一份。
